# Image Attributions

This directory contains third-party logos and artwork used to identify
mod loaders and other software that Histolauncher integrates with.
Histolauncher is not affiliated with or endorsed by any of the projects
listed below; their logos are used solely to identify the corresponding
loader inside the launcher UI.

If you are a rights holder and would like a logo removed or replaced,
please open an issue on the Histolauncher repository or contact `histolauncher@gmail.com`.
